# Aplicação WEB para a Escola Facs

Projeto feito com a biblioteca React.js + TypeScript, afim de obtenção de nota.

## Instalação

Clone o projeto na sua máquina e faça:

```bash
  npm install
  npm run dev
```

## Stack utilizada

**Front-end:** React, Axios, Material-UI, Yup
